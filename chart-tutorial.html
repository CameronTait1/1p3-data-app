1. Introduction to Chart.js

Chart.js is a JavaScript library that allows you to create responsive charts. In this web app a bar chart and pie chart were created, however there are 3 you can use. You can choose from various chart types including bar, line, pie, and more.

  • Bar Chart: Displays data in rectangular bars, useful for comparing categories.
  • Line Chart: Shows data points connected by a line, ideal for trend analysis.
  • Pie Chart: Represents data as segments of a circle, useful for showing proportions.

Include Chart.js in your HTML file by adding the following script tag:

<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

Hereâ€™s how to create a simple bar chart with Chart.js:

<script>
const ctx = document.getElementById('myBarChart').getContext('2d');
new Chart(ctx, {
    type: 'bar',
    data: {
        labels: ['Red', 'Blue', 'Yellow'],
        datasets: [{
            label: 'Number of Votes',
            data: [12, 19, 3],
            backgroundColor: [
                'rgba(255, 99, 132, 0.2)',
                'rgba(54, 162, 235, 0.2)',
                'rgba(255, 206, 86, 0.2)'
            ],
            borderColor: [
                'rgba(255, 99, 132, 1)',
                'rgba(54, 162, 235, 1)',
                'rgba(255, 206, 86, 1)'
            ],
            borderWidth: 1
        }]
    },
    options: {
        scales: {
            y: {
                beginAtZero: true
            }
        }
    }
});
</script>

Customize the chart by adjusting properties such as labels, data, and colors.

3. Creating a Line Chart

To create a line chart, use the following example:


<canvas id="myLineChart" width="400" height="200"></canvas>

<script>
const ctx = document.getElementById('myLineChart').getContext('2d');
new Chart(ctx, {
    type: 'line',
    data: {
        labels: ['January', 'February', 'March', 'April'],
        datasets: [{
            label: 'Monthly Sales',
            data: [65, 59, 80, 81],
            fill: false,
            borderColor: 'rgb(75, 192, 192)',
            tension: 0.1
        }]
    },
    options: {
        scales: {
            y: {
                beginAtZero: true
            }
        }
    }
});
</script>

Modify the dataset and options to fit your data and desired appearance.
